# streaming-site-structure-analysis
该项目围绕在线影视平台结构分析整体概览展开整理，对首页导航层级逻辑、分类标签体系以及播放页信息结构进行系统归档。通过结构化文字记录建立资料库，用于分析不同影视平台在结构分析规划方面的变化趋势。仓库仅保存公开页面资料。
